# Where to Win — India GCC Location Strategy Dashboard

An interactive Power BI dashboard for evaluating and shortlisting 
Tier-2 Indian cities as potential Global Capability Centre locations.

## Overview
This dashboard provides a structured, data-driven framework for 
evaluating 20 Tier-2 Indian cities across 25 metrics spanning five 
strategic dimensions. Priority weights are fully configurable by GCC 
function type, enabling organisations to generate location 
recommendations aligned to their specific business needs.

## Dashboard Pages
1. About — Project overview, objective and data credits
2. Decision Engine — Configure priority weights by GCC type
3. Leaderboard — City attractiveness scorecard
4. City Deep Dive — Comparative dimension analysis
5. India Overview — Geographic distribution map
6. Sensitivity Analysis — Scenario robustness testing

## Five Strategic Dimensions
| Dimension | Metrics Included |
|---|---|
| Talent Availability | % S&T graduates, tech talent density, literacy, college density |
| Cost Competitiveness | Average salary, office rental, cost of living |
| Infrastructure Readiness | Metro, flights, internet speed, SEZ area, EoDB rank |
| Tech Ecosystem Depth | Tech business density, education & medical ecosystem indices |
| Employee Liveability | Crime index, pollution index, cost of living |

## Data Source
Primary data sourced from "Emerging Technology Hubs of India" — 
Deloitte Touche Tohmatsu India LLP & NASSCOM, August 2023.

## How to Use
1. Download GCC_Dashboard.pbix
2. Open in Power BI Desktop (free)
3. Navigate to Decision Engine
4. Select your GCC type or configure custom weights
5. Rankings update in real time
